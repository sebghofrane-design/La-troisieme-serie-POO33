# La-troisieme-serie-POO33
OOP TP3 - Object Oriented Programming Project

This project contains 4 exercises:

1. Bank Account System
   - Deposit / Withdraw / Balance management

2. Person & Student
   - Inheritance
   - Pass/Fail system

3. Employee & Manager
   - Salary + Bonus calculation

4. Shapes
   - Abstract class
   - Rectangle & Circle implementation

Developed using Java (Eclipse).
